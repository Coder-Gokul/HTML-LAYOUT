  • Form

    • Attributes:
      • action, autocomplete="off", method=""
    • Input type:
      • text (attribute: required, readyonly, disabled, ), password, date, email, datetime-local, file, hidden, number, color, range, radio (for multiple radio button common name attribute should be given), checkbox, search, tel, time, url, week, image, select & option (attribute: multiple), textarea (attribute: cols=”” rows=””), reset
    • Example:
        
        <form action=”#”>
            <fieldset>
            <legend> Title </legend>
                <label for=”test”> Name </label>
                <input type=”text” id= “test” placeholder=”enter here” title=”Enter Username” >
                <input type=”email” required readonly disabled>
                <input type=”file”  multiple >
                <input type=”number”  min=”1” max=”10” >
                <input type=”hidden”  value=”Secret value to check the human submission”>
                <input type=”submit” value=”Login”>
                <button> Login </button>
            </fieldset>
        </form>
